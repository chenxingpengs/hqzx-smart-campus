# 晚午托考勤小程序用户手册

## 一、简介

晚午托考勤小程序是一款专为学校教师设计的考勤管理工具，支持午托和晚托两个时段的考勤记录、查询和排班管理。

### 适用人群
- 班主任：管理本班学生考勤
- 值班教师：管理排班班级考勤
- 管理员：查看全校考勤数据

---

## 二、快速开始

### 2.1 登录
1. 打开小程序，首次使用需点击【登录】按钮
2. 微信用户将自动授权登录
3. 登录成功后显示用户信息

### 2.2 首页导航
首页顶部显示当前日期，下方有三个功能标签：
- **我的考勤**：查看考勤总览和班级列表
- **考勤查询**：按条件查询历史考勤记录
- **排班**：管理个人排班安排

---

## 三、我的考勤

### 3.1 考勤总览
进入【我的考勤】页面，可以看到：

| 区域 | 说明 |
|------|------|
| 考勤进度条 | 显示今日整体考勤完成百分比 |
| 今日待完成 | 显示今日未完成考勤的班级数量 |
| 月度统计 | 显示本月总考勤班级数、已完成数、待完成数 |

### 3.2 我的班级（班主任专属）
如果您是班主任，页面顶部会显示【我的班级】卡片：

| 字段 | 说明 |
|------|------|
| 班级名称 | 您所负责的班级 |
| 时段 | 午托/晚托 |
| 应到人数 | 班级学生总数 |
| 实到人数 | 今日已到学生数 |
| 缺勤人数 | 今日未到学生数 |
| 请假人数 | 今日请假学生数 |

**操作按钮**：
- **去考勤**：进入本班考勤详情页
- **查看本班详情**：查看本班考勤详细信息

### 3.3 其他排班班级
页面下方显示您负责的其他排班班级列表：

**班级卡片信息**：
- 班级名称、年级
- 应到学生人数
- 时段（午托/晚托）
- 排班时间
- 今日考勤状态

**快捷操作**：
- 点击【全勤】按钮：一键标记全班出勤
- 点击班级卡片：进入详细考勤页面

### 3.4 状态说明

| 状态 | 说明 |
|------|------|
| ✅ 已完成 | 该班级今日考勤已完成 |
| ⏳ 待完成 | 该班级今日考勤未完成 |
| ⚠️ 信息缺失 | 班级信息不完整，需在终端补充 |
| 🔴 数据异常 | 考勤数据存在问题（如实到>应到） |

### 3.5 提醒功能
系统会自动推送以下提醒：

| 提醒类型 | 触发条件 |
|----------|----------|
| 考勤即将截止 | 距离考勤截止时间≤30分钟 |
| 新增请假 | 有新的请假申请待处理 |
| 昨日未完成 | 昨日有班级未完成考勤 |
| 批量待处理 | 当前时段有多个班级未考勤 |

---

## 四、考勤查询

### 4.1 查询条件
进入【考勤查询】页面，可按以下条件筛选：

| 条件 | 说明 |
|------|------|
| 查询日期 | 选择要查询的日期 |
| 年级筛选 | 初一/初二/初三/全部 |
| 班级名称 | 输入班级名称关键词搜索 |

### 4.2 查询结果
点击【执行查询】后，显示符合条件的考勤记录：

- 班级名称
- 时段（午托/晚托）
- 排班时间
- 应到人数
- 已到人数
- 请假人数

---

## 五、排班管理

### 5.1 查看排班
进入【排班】页面，可以看到：

1. **日历视图**：显示当月日历，有排班的日期会显示标记
2. **我的排班列表**：显示所有排班安排

### 5.2 新增排班
1. 在日历上选择值班日期
2. 选择值班时段（午托/晚托）
3. 选择年级和班级
4. 点击【确认排班】

### 5.3 取消排班
1. 在【我的排班安排】列表中找到要取消的排班
2. 点击【取消排班】按钮
3. 确认取消

> **注意**：固定时段排班（如晚托固定排班）无法取消

### 5.4 排班状态

| 标记 | 说明 |
|------|------|
| 🟢 有空位 | 该时段可排班 |
| 🔴 已排班 | 您已在该时段排班 |
| ⚫ 无空位 | 该时段已满，不可排班 |
| 📌 固定排班 | 系统固定分配，不可修改 |

## 七、常见问题

### Q1：考勤数据不显示怎么办？
1. 下拉刷新页面
2. 检查网络连接
3. 重新登录

### Q2：班级信息缺失如何处理？
班级信息（应到人数、年级等）需要在班级终端设备上完善，小程序端无法修改。

### Q3：误操作如何撤销？
- 全勤标记：进入班级详情页重新编辑
- 排班取消：重新添加排班即可

### Q4：为什么有些时段无法选择？
- 该时段已有您的其他排班（冲突）
- 该时段已满员
- 该时段为固定排班

### Q5：考勤截止时间是什么？
每个时段都有考勤截止时间，超过截止时间后仍可补录，但会有提醒。

## 八、操作流程图

### 考勤流程
```
进入首页 → 我的考勤 → 选择班级 → 查看学生列表 → 标记出勤/请假 → 提交
```

### 排班流程
```
进入排班 → 选择日期 → 选择时段 → 选择班级 → 确认排班
```
